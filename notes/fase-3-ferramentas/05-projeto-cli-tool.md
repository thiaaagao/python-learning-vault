## Opções de Projeto

Escolha **um** dos projetos abaixo:

### Opção A: Gerenciador de Senhas CLI
Comandos:
- `python passman.py add --site exemplo --user admin --senha 123`
- `python passman.py get --site exemplo`
- `python passman.py list`
- `python passman.py generate --length 16`

### Opção B: Ferramenta de Análise de Rede
Comandos:
- `python nettool.py scan --host 192.168.1.0/24`
- `python nettool.py resolve --host example.com`
- `python nettool.py whois --ip 8.8.8.8`
